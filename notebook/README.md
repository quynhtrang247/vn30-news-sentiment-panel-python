# Notebooks

This folder contains the Jupyter Notebook used in the project **Impact of News Sentiment on Stock Returns: Evidence from VN30**.

## File
- `project_code.ipynb`: main notebook for news collection, data preprocessing, panel-data construction, model estimation, diagnostics, and visualization

## Main Tasks Covered
- Collecting financial news
- Retrieving market data
- Merging sentiment and stock data
- Building a panel dataset
- Estimating panel regression models
- Producing charts and model outputs
